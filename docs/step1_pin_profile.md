# Step 1 — Pin showcase on your GitHub profile

Do this **after** the showcase repo is **public** ([Step 0.5](distribution_plan.md#step-05--flip-showcase-to-public-when-vitrin-is-finalized)).

## Pin `project-a-showcase`

1. Open **https://github.com/umut-yasin-yilmaz** while logged in.
2. Click **Customize your pins** (or **Edit profile** → pinned repositories section, depending on UI).
3. Select **`project-a-showcase`** (up to 6 repos).
4. Save.

Private repos **do not** show as pins to other users; once public, the pin is visible to visitors.

## Optional

- **Profile → Settings → Profile:** set **Bio** (short role + stack) and **URL** (showcase repo link or future Steam/Itch).
